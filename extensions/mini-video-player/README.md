# Mini Video Player Extension

This browser extension shrinks any video on a page into a floating mini player with custom controls.

## Features

- Click the **Mini** button on any video to open it in a floating window.
- Adjustable volume, skip forward/backward 10 seconds, and close the mini player.
- Rounded corners and subtle shadow for a modern look.

## Installation

1. Open your browser's extensions page.
2. Enable developer mode and load this folder as an unpacked extension.

The extension will add a **Mini** button to each video element on pages you visit.
